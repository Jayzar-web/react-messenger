<h2>🛠️ Installation Steps:</h2>

<p>1. Clone repository</p>

```
git clone git@github.com:Jayzar-web/react-messenger.git && cd react-messenger
```

<p>2. dependency installation</p>

```
npm install
```

<p>3. start</p>

```
npm run dev
```
